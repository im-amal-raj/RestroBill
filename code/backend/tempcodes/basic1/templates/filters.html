Filters in jinja2 (flask framework)

cannot use .upper() in html with jinja2
for that the jinja2 filters came. syntax : var | filter_name

Example for upper(),lower(),replace() and make custom filter

normal : {{ text }}

Uppercase : {{ text | upper }}

Lowercase : {{ text | lower }}

Replace l with L : {{ text | replace('l','L') }}

custome :-

reverse_string : {{ text | reverse_string }}

repeat default=2 : {{ text | repeat }}

repeat with 5 : {{ text | repeat(5) }}

alternate_case (upper-lower-upper-etc) : {{ text | alternate_case }}
